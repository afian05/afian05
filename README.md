# 🪙 NaifCoin (NFC)

**The Power of Blockchain. The Stability of Gold.**

NaifCoin is a gold-backed utility token built on BNB Smart Chain, developed by **Naif Global Indonesia (NGI)**. It bridges the stability of physical gold with the accessibility and transparency of blockchain technology, targeting the Indonesian retail market.

## 🔧 Core Features

- **Staking Rewards:** 8% APY with a first-stake bonus for early participants
- **Treasury Mechanism:** 1% transaction fee routed to a treasury reserved for physical gold acquisition
- **Gold-Backed Marketplace (upcoming):** Smart contract powered by a Chainlink XAU/USD price oracle, tying token utility directly to real-time gold prices

## ⚖️ Regulatory Approach

NaifCoin is being developed with attention to Indonesian financial regulations, including:
- **POJK 23/2025** — Crypto asset framework
- **POJK 17/2024** — Gold-backed instrument (Kegiatan Usaha Bulion) framework

The team maintains an active compliance dialogue with Indonesia's OJK regulator.
